## 目录结构说明

### 核心目录

#### api/
- **config.js**: 集中管理API基础地址和路径配置，支持本地开发环境和Apifox Mock环境切换

#### components/
- **course-view/**: 课程列表视图，包含课程分类、状态展示和进度条
- **home-view/**: 首页视图，包含Banner、导航宫格、名言展示和课程列表
- **mine-view/**: 个人中心视图，包含用户信息和功能入口

#### pages/
- **CourseDetail/**: 课程详情页面，展示课程详细信息和学习进度
- **Login/**: 登录相关页面，包含登录主页面和用户信息补全页面
- **Main/**: 主页面，集成底部导航栏和各功能模块入口
- **volunteer/**: 志愿者相关页面，包含作业管理、统计分析等功能

#### static/
- **icons/**: 功能图标集合，包含证书、课程、考试等图标
- **logo.png**: 项目品牌Logo

### 配置文件

- **App.vue**: 应用入口组件，定义全局生命周期和样式
- **main.js**: JavaScript入口文件，支持Vue 2和Vue 3双版本
- **pages.json**: 页面路由配置，定义页面路径和导航栏样式
- **manifest.json**: 应用配置文件，包含应用名称、版本等信息
- **uni.scss**: 全局样式文件，定义主题色和通用样式变量

## 项目技术概述

### 技术栈

| 技术/框架 | 版本 | 用途 |
|---------|------|------|
| UniApp  | 最新 | 跨平台应用开发框架 |
| Vue     | 2/3  | 前端视图框架 |
| JavaScript | ES6+ | 开发语言 |
| SCSS    | 最新 | CSS预处理器 |
| uni-ui  | 最新 | UniApp官方组件库 |

### 核心功能模块

#### 1. 用户认证系统
- **登录功能**: 支持账号密码登录
- **信息补全**: 首次登录后引导用户完善个人信息
- **Token管理**: 使用本地存储管理认证令牌
- **身份验证**: 基于Token的API请求头认证

#### 2. 课程管理系统
- **课程分类**: 正在学习、历史课程、已结业
- **课程详情**: 包含课程介绍、进度跟踪、作业管理
- **学习状态**: 实时展示课程学习进度
- **状态管理**: 课程状态（进行中、已完成、已过期）

#### 3. 志愿者管理系统
- **作业管理**: 提交、查看、统计作业
- **数据统计**: 志愿者活动数据可视化
- **优秀作业**: 展示优秀作业案例
- **个人中心**: 志愿者个人信息管理

#### 4. 首页功能
- **Banner展示**: 项目理念和宣传内容
- **导航宫格**: 核心功能模块入口
- **名言展示**: 传统文化名言警句
- **热门课程**: 推荐课程列表

### 技术特点

#### 1. 跨平台兼容性
- 基于UniApp框架，支持多端发布
- 同时支持Vue 2和Vue 3语法
- 适配不同屏幕尺寸的响应式布局

#### 2. 组件化开发
- 公共组件复用（课程、首页、个人中心）
- 页面级组件拆分（课程详情子组件）
- 模块化API配置管理

#### 3. 性能优化
- 防抖处理（登录请求）
- 异步请求管理
- 本地缓存策略
- 条件编译支持多平台

#### 4. 用户体验
- 加载状态反馈
- 错误处理与提示
- 平滑过渡动画
- 响应式交互反馈

#### 5. 安全机制
- Token认证机制
- 请求头授权
- 表单验证
- 输入过滤

### 网络请求架构

- **统一API配置**: 集中管理接口地址和路径
- **Token认证**: 基于请求头的身份验证
- **错误处理**: 统一的网络错误捕获和提示
- **环境切换**: 支持本地开发和Mock数据环境

### 本地存储管理

- **Token存储**: 认证令牌持久化
- **用户信息**: 本地缓存用户基本信息
- **状态管理**: 页面状态和用户偏好设置

### 路由管理

- **页面导航**: 基于pages.json的路由配置
- **页面跳转**: 支持多种跳转方式（reLaunch、redirectTo等）
- **参数传递**: 页面间数据通信
- **权限控制**: 未登录状态自动跳转到登录页

## 开发环境配置

### 必要依赖
- Node.js
- HBuilderX（推荐）或其他UniApp支持的IDE
- Vue CLI（可选）

### 启动项目
1. 克隆代码库
2. 使用HBuilderX打开项目
3. 选择运行平台（小程序、H5、App等）
4. 启动开发服务器

### API配置
修改 `api/config.js` 文件中的 `baseUrl` 配置，切换开发环境：

```javascript
// 本地开发环境
baseUrl: 'http://127.0.0.1:8080'

// 或使用Apifox Mock环境
// baseUrl: 'http://127.0.0.1:4523/m1/7779396-7525646-default'
```
